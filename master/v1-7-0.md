# v1.7.0 更新日志

> {tip} 本次版本没有包含不兼容改动，适合所有用户升级。

## 升级方法

```bash
composer update dcat/laravel-admin
```

本次版本需要重新发布**静态资源**以及**语言包**，然后按`Ctrl`+`F5`**清除浏览器缓存**！
```bash
php artisan admin:publish --assets --force
php artisan admin:publish --lang --force
```
